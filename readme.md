# XY News
  App gathers World news from CNN and BCC

## About 
app uses ejs as the view template, backed with express, news Api fetched with axios and moment embedded the time.

## Built With
  - ejs
  - express
  - axios
  - moment
  - dotenv
  - news API
  - bootstrap 4
